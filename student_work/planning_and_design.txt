=========================
APP PLANNING & DESIGN
=========================

1. PROJECT NAME & THEME
-------------------------
- MyPrivateLife
- pink / yellow color scheme
- To access the app you need a "key" or a password

2. CORE FUNCTIONALITY
-------------------------
- Diary type function, take personal notes that are 100% private
- customization of diary optional

3. SECRET OR PERSONALIZED CONTENT

-------------------------
- What content will users see after logging in? You see a diary that is randomized (Can be hypothetically customized later)
- Will this content change depending on who is using the app (admin, user, special user?) no (unless customized hypothetically)

4. USER EXPERIENCE (UX) DESIGN
-------------------------
- How should your app look and feel? more cozy like a small diary
- How will users navigate between pages? arrow keys to flip pages, shift + enter to start typing
once typing arrow keys behave as they do on writing software, use shift + enter to
exit typing mode

5. SECURITY FEATURES
-------------------------
- What password and authentication rules will you enforce?  
- How will you protect secret pages from unauthorized access?
The whole app is secret, you set a password at first and if you get it wrong upon next
login five times in a row, you will be put on a 5 minute time out before you can log in
again.

6. OPTIONAL CREATIVE ENHANCEMENTS
-------------------------
- What customization or fun features could users enjoy? fonts, sizes, bold/italic/underline
- Are there interactive elements, like games, quizzes, or badges, you could include? stickers
